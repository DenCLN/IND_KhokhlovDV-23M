## Заявленное задание:
Модуль для учёта персональных тренировок. Система должна позволять тренерам создавать программы тренировок для клиентов.            
Тема звучит так - "Система управления персональными тренировками в фитнес-центре"
Будет состоять из четырёх коллекций.
1. - Клиент, где будут храниться данные о клиенте.
Операции связанные с этой коллекцией: Создание, удаление клиента, обновление данных, получение по id, получение
списка всех клиентов
2. - Тренер, где также будут храниться данные о тренере, специализации.
Операции: Создание, удаление тренера, обновление данных, получение по id, специализации, получение списка всех
тренеров
3. - План тренировок, будет включать в себя название, описание, создателя (тренер), цель, сложность.
Операции: Создание, редактирование, удаление плана, добавление упражнений, получение по id, получение планов для
определённого клиента, поиск по сложности, клонирование.
4. - Упражнение, где будет храниться его описание, фото, видео или gif содержание с техникой выполнения упражнения,
которое можно будет скачать.
Операции: создание, обновление, удаление, получение по id, поиск по мышечной группе, получение
всех упражнений

## Функционал системы

Система предоставляет четыре основных модуля:

### 1. Управление клиентами (`/clients`)
- Создание, просмотр, обновление и удаление клиентов
- Хранение персональных данных: имя, фамилия, email, телефон, дата рождения, заметки

### 2. Управление тренерами (`/trainers`)
- Создание, просмотр, обновление и удаление тренеров
- Хранение информации о специализации, опыте работы
- Фильтрация тренеров по специализации

### 3. Управление планами тренировок (`/plans`)
- Создание планов тренировок с указанием цели, сложности (EASY, MEDIUM, HARD)
- Привязка планов к конкретным клиентам и тренерам
- Добавление упражнений в план
- Клонирование существующих планов
- Фильтрация по сложности и клиенту

### 4. Библиотека упражнений (`/exercises`)
- Создание справочника упражнений
- Классификация по группам мышц (Ноги, Грудь, Спина, Плечи, Руки, Ягодицы, Пресс)
- Хранение ссылок на фото, видео и технику выполнения
- Фильтрация упражнений по группе мышц

## Структура проекта

### Корневая директория

- **`prisma/`** - схема базы данных и конфигурация Prisma
- **`test/`** - тесты приложения
- **`API_ENDPOINTS.md`** - документация по API эндпоинтам

### Директория `src/`

#### **`clients/`**
Модуль управления клиентами:
- **`clients.controller.ts`** - контроллер, обрабатывающий HTTP-запросы (GET, POST, PATCH, DELETE)
- **`clients.service.ts`** - бизнес-логика работы с клиентами
- **`clients.module.ts`** - модуль, объединяющий контроллер и сервис
- **`dto/`** -  для валидации входных данных:
  - `create-client.dto.ts` - DTO для создания клиента
  - `update-client.dto.ts` - DTO для обновления клиента

#### **`trainers/`**
Модуль управления тренерами (структура аналогична модулю clients):
- **`trainers.controller.ts`** - обработка запросов с поддержкой фильтрации по специализации
- **`trainers.service.ts`** - бизнес-логика
- **`trainers.module.ts`** - модуль
- **`dto/`** - DTO для создания и обновления тренеров

#### **`plans/`**
Модуль управления планами тренировок:
- **`plans.controller.ts`** - обработка запросов, включая добавление упражнений и клонирование планов
- **`plans.service.ts`** - бизнес-логика работы с планами
- **`plans.module.ts`** - модуль
- **`dto/`** - DTO для создания и обновления планов

#### **`exercises/`**
Модуль библиотеки упражнений (структура аналогична другим модулям):
- **`exercises.controller.ts`** - обработка запросов с фильтрацией по группе мышц
- **`exercises.service.ts`** - бизнес-логика
- **`exercises.module.ts`** - модуль
- **`dto/`** - DTO для создания и обновления упражнений

### Директория `prisma/`

- **`schema.prisma`** - схема базы данных, определяющая модели:
  - `Client` - модель клиента
  - `Trainer` - модель тренера
  - `TrainingPlan` - модель плана тренировки
  - `ExerciseLibrary` - модель упражнения в библиотеке
  - `Difficulty` - перечисление уровней сложности (EASY, MEDIUM, HARD)
  - `Exercise` - тип данных для упражнений в плане тренировки
