## Клиенты (`/clients`)

| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/clients` | Создать клиента |
| GET | `/clients` | Получить всех клиентов |
| GET | `/clients/:id` | Получить клиента по ID |
| PATCH | `/clients/:id` | Обновить клиента |
| DELETE | `/clients/:id` | Удалить клиента |

**Поля для создания клиента:**
- `firstName` (обязательно)
- `lastName` (обязательно)
- `email` (обязательно, уникальный)
- `phone` (опционально)
- `dateOfBirth` (опционально, формат: YYYY-MM-DD)
- `notes` (опционально)

## Тренеры (`/trainers`)

| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/trainers` | Создать тренера |
| GET | `/trainers` | Получить всех тренеров |
| GET | `/trainers?specialization=Йога` | Получить тренеров с фильтром |
| GET | `/trainers/:id` | Получить тренера по ID |
| PATCH | `/trainers/:id` | Обновить тренера |
| DELETE | `/trainers/:id` | Удалить тренера |

**Поля для создания тренера:**
- `firstName` (обязательно)
- `lastName` (обязательно)
- `email` (обязательно, уникальный)
- `phone` (опционально)
- `specialization` (обязательно)
- `experienceYears` (обязательно, число 0-80)
- `about` (опционально)

**Query параметры для GET /trainers:**
- `specialization` - фильтр по специализации

## Планы тренировок (`/plans`)

| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/plans` | Создать план тренировки |
| GET | `/plans` | Получить все планы |
| GET | `/plans?difficulty=EASY` | Фильтр по сложности |
| GET | `/plans?clientId=xxx` | Фильтр по клиенту |
| GET | `/plans/:id` | Получить план по ID |
| PATCH | `/plans/:id` | Обновить план |
| POST | `/plans/:id/exercises` | Добавить упражнения в план |
| POST | `/plans/:id/clone` | Клонировать план |
| DELETE | `/plans/:id` | Удалить план |

**Поля для создания плана:**
- `title` (обязательно)
- `description` (опционально)
- `goal` (обязательно)
- `difficulty` (обязательно: EASY, MEDIUM, HARD)
- `creatorId` (обязательно, ID тренера)
- `clientId` (опционально, ID клиента)
- `exercises` (опционально, массив упражнений)

**Структура упражнения в плане:**
- `name` (обязательно)
- `description` (опционально)
- `sets` (опционально, число)
- `reps` (опционально, число)
- `durationSec` (опционально, число)

**Query параметры для GET /plans:**
- `difficulty` - фильтр по сложности (EASY, MEDIUM, HARD)
- `clientId` - фильтр по клиенту

## Упражнения - Библиотека (`/exercises`)

| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/exercises` | Создать упражнение в библиотеке |
| GET | `/exercises` | Получить все упражнения |
| GET | `/exercises?muscleGroup=Ноги` | Фильтр по группе мышц |
| GET | `/exercises/:id` | Получить упражнение по ID |
| PATCH | `/exercises/:id` | Обновить упражнение |
| DELETE | `/exercises/:id` | Удалить упражнение |

**Поля для создания упражнения:**
- `name` (обязательно)
- `description` (опционально)
- `muscleGroup` (обязательно, например: "Ноги", "Грудь", "Спина", "Плечи", "Руки", "Ягодицы", "Пресс")
- `photoUrl` (опционально, URL)
- `videoUrl` (опционально, URL)
- `techniqueUrl` (опционально, URL)

**Query параметры для GET /exercises:**
- `muscleGroup` - фильтр по группе мышц
